One-Time Setup (≈ 3 min)

Step 1

Open console.cloud.google.com, sign in, tap New Project → name it DeliveryTracker → Create.

Step 2

Go to APIs & Services → Library, search Google Drive API, tap Enable.

Step 3

Go to OAuth consent screen → External → Create. Fill in app name & your email. Save through all steps.

Step 4

Go to Credentials → Create Credentials → OAuth client ID → Web application. Under Authorized JavaScript origins add null. Tap Create.

Step 5

Copy the Client ID (ends in .apps.googleusercontent.com) and paste below.
